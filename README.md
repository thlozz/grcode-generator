# QR Code Generator API 🚀

Uma API robusta desenvolvida em **Java 21/25** utilizando **Spring Boot**, focada em performance e simplicidade para geração de QR Codes dinâmicos.

## 🛠️ Tecnologias e Ferramentas
* **Linguagem:** Java (JDK 21+)
* **Framework:** Spring Boot 3.x
* **Biblioteca de Core:** Google ZXing (Zebra Crossing)
* **Gerenciador de Dependências:** Maven
* **Ambiente de Desenvolvimento:** Linux (Debian/ChromeOS)

## 📋 Funcionalidades
* Endpoint RESTful para geração de QR Code via URL.
* Retorno direto em formato de imagem (PNG).
* Lógica desacoplada facilitando a manutenção e testes futuros.
